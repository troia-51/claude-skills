# Evals — CFP Study
> Pass/fail checks for output quality. 每次教學 session 之後用呢個 checklist 驗收。

## Teaching Method

- [ ] 先問學生已有知識，再開始解釋（Socratic Method）
- [ ] 解釋長度約 200 words，唔會太長
- [ ] 解釋後有 1-2 個理解檢查問題
- [ ] 根據學生回答調整教學方式（adaptive follow-up）

## Content Accuracy

- [ ] 所有稅率、限額、phase-out range 嘅數字都經網上搜索驗證
- [ ] 唔會純靠 training data 猜答案
- [ ] 引用權威來源（IRS.gov、CFP Board 等）
- [ ] 計算題經雙重驗證

## Topic Coverage

- [ ] 解釋符合 CFP exam 嘅 domain 結構（A-H）
- [ ] 優先處理高權重 domain（Retirement 18%, Investment 17%）
- [ ] 連結概念到 CFP exam 實際應用場景

## Session Tracking

- [ ] 每個 session 有 `/sessions/YYYY-MM-DD/session-notes.md` 記錄
- [ ] `/progress/cfp-study-tracker.md` 已更新（進度、gaps、mastered topics）
- [ ] Session notes 包含：學生問題、理解水平、identified gaps、mastered topics

## Error Handling

- [ ] 學生指出錯誤時立即承認並搜索驗證
- [ ] 更正後清楚展示正確答案同來源
- [ ] 唔會為錯誤答案辯護

## Language & Tone

- [ ] 語氣友善、非批判性
- [ ] 唔會用未解釋嘅專業術語
- [ ] 鼓勵學生提問和探索
